# Portfolio 95
 
> This is my personal portfolio with a Window-95 inspired design

---
 
## Technical stack
 
- HTML5 / CSS3 / JavaScript (Vanilla)
- Web Components API (`<app-window>`)
- Font : [W95FA] https://www.dafont.com/w95fa.font
- No framework — No dependencies

---
 
## Project structure
 
```
portfolio-95/
├── index.html
├── css/
│   ├── style.css        # General styles
│   └── styleFen.css     # Windows Styles
├── js/
│   └── main.js          # Web components
├── img/
│   ├── icons/           # Icons Windows-95
│   └── exemplePortfolio/ # GIF projects
└── font/
    └── W95FA/           # Windows-95 font
```
 
---
 
## Installation
 
No dependencies — This project run with native HTML/CSS/JS.
 
```bash
git clone https://github.com/sn1675/portfolio.git
cd portfolio
# Then open index.html in you web browser or run it on your local server
```
