Le Demineur

Langage : Python

Librairie(s) : Tkinter

Description : Qui ne se souvient pas ? Enfant, on jouait aux jeux installés directement sur l’OS Windows, le démineur en faisait partie.
Voici mon premier vrai gros projet, un démineur fait entièrement en Python.
Ce démineur propose plusieurs modes de jeu, comme facile, expert ou encore aléatoire. Ils permettent de changer le nombre de cases et de mines dans le jeu pour rajouter du piment à vos parties.

GaphicalSort

Langage : Python

Librairie(s) : Pygame

Description : Si vous avez déjà codé quelques projets, il vous est sûrement arrivé de tomber sur des algorithmes de tri,
ou encore mieux, sur des vidéos de "sort" sur YouTube dans lesquelles on voit le fonctionnement de différents algorithmes de tri.
Voici donc mon projet GraphicalSort qui, comme son nom l'indique, permet de voir graphiquement un tri se réaliser.
Sur la vidéo ci-dessus, vous pouvez voir l'algorithme de tri par insertion qui trie une liste de nombres mélangés aléatoirement
de 1 à 100.
